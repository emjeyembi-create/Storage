Mark's Portfolio – Final Project

This repository contains my personal portfolio website created as part of my Final Project in the BS Computer Science program.  
The website showcases my background, achievements, skills, projects, and contact information in a single-page layout.

GitHub Repository: https://github.com/emjeyembi-create/Storage  


Overview
This portfolio is a single-page website built using HTML and Bootstrap 5, styled with a futuristic cyber-themed design using the Orbitron font and an animated background.


The website includes:
Home / Hero – Introduction, profile image, and contact button  
About Me – Personal background, interests, hobbies  
Achievements – Competitions, awards, and personal interests  
Projects – Figma prototypes: Food Ordering App and Real Estate Website  
Skills – Programming languages, tools, and IDEs with progress bars  
Contact – Functional email form using FormSubmit plus social media links


Features
Responsive layout using Bootstrap 5  
Sticky navigation bar  
Animated background  
Orbitron font styling  
Icons from Devicon and Font Awesome  
Contact form using FormSubmit to send messages directly to my email
