Overview

This project is a single-page portfolio website built using HTML, Bootstrap 5, and Laravel Blade components.
It is styled with a futuristic cyber-themed design using the Orbitron font and animated background effects.

The system supports user authentication and project management through CRUD operations for authorized users.

System Features

Public Features
Responsive single-page layout
Sticky or fixed navigation bar
Animated cyber-themed background
Orbitron font styling
Icons from Devicon and Font Awesome
Contact form that sends messages directly to email via FormSubmit

Authentication Features
User login and registration
Secure authentication using Laravel
Conditional navigation links based on authentication status

CRUD Functionality for Authenticated Users
Create new projects
View existing projects
Update project information
Delete projects
Admin-only project management interface

These features demonstrate the implementation of CRUD operations, authentication, and access control.
